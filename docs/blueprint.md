# **App Name**: Baba Sai Eats

## Core Features:

- Header Branding: Displays the restaurant's name and tagline prominently at the top of the page.
- Categorized Menu: Presents food items grouped under Soup, Noodles, Manchurian, Bhel, Paneer, and Rice categories.
- Dish Presentation: Each dish includes a name, and optional pricing details, arranged neatly.
- Sticky Navigation Tabs: Category tabs stick to the top for easy access, adapting as you scroll, for easy filtering and navigation between food categories.
- Footer Information: Footer displays contact info, order info and accepted payment methods for customer convienience.
- WhatsApp Contact Button: A floating WhatsApp button stays in view for quick customer support.
- Smart Suggestion Tool: AI-powered tool to suggest menu items to customers based on factors like order history, dietary restrictions, and popularity.

## Style Guidelines:

- Primary color: Saturated yellow-gold (#D4AF37) to convey the richness and warmth of home-style Chinese cuisine.
- Background color: Very light, desaturated yellow (#F8F5EC), of the same hue as the primary color, which gives a clean and inviting backdrop that won't overpower the food photos.
- Accent color: A warm orange (#E07A5F), analogous to the yellow-gold, is used to highlight calls to action.
- Headline font: 'Playfair' serif with elegant lines, suggesting high quality. Body font: 'PT Sans' sans-serif, for easy reading.
- Mobile-first approach, using a clear, single-column layout for readability and intuitive navigation.
- Use clean, minimalist icons for payment methods and contact options.
- Subtle transition animations when switching between menu categories for a smooth user experience.
